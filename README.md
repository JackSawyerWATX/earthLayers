# Earth's Layers Visualizer

A simple Python visualization of Earth's internal structure using the built-in `turtle` graphics library. The program draws a cross-section of Earth showing the three primary layers — crust, mantle, and core — each rendered as a filled circle with a distinct color and label.

## Technologies Used

- **Python 3.x** — The program is written in standard Python with no external dependencies.
- **turtle** — A Python standard library module for basic vector graphics and drawing. It comes pre-installed with Python, so no additional packages are required.

## How to Run

1. **Make sure Python is installed.** You can verify by running:
   ```bash
   python --version
   ```
   If Python is not installed, download it from [python.org](https://www.python.org/downloads/).

2. **Save the script.** Copy the code into a file named `earth_layers.py`.

3. **Run the script:**
   ```bash
   python earth_layers.py
   ```

4. A window will open displaying the Earth's layers diagram. Close the window when you're done.

> **Note:** The `turtle` module requires a display environment. It will not run in headless terminals or some online code editors. Run it locally on your machine.
